# Bank-Loan-Application-Excel-Excel-Dashboard
Bank Loan Application Dashboard provides a clear overview of loan performance using interactive visuals. It tracks total applications, funded and received amounts, interest rates, loan quality, and trends by purpose, term, state, and month to support data-driven lending decisions.

## Project Objevtive
- To Analyze overall loan application and performance metrics
- To Evaluate loan quality by comparing good and bad loans
- To Identify trends by loan purpose, term, and employment length
- To Track monthly and state-wise loan application patterns
- To Support data-driven credit risk and lending decisions
- To Monitor interest rates and debt-to-income ratios
- To Understand borrower behavior and demand patterns
- To Improve loan portfolio management and decision-making
